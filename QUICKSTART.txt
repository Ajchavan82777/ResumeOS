FIRST-TIME SETUP  (run once)
═══════════════════════════════════════════════════════════════════

STEP 1 — Install Node.js v18+
  https://nodejs.org/en/download

STEP 2 — Create Supabase project
  1. https://app.supabase.com → create free project
  2. SQL Editor → New query
  3. Paste ALL of: backend/src/db/schema.sql → Run
     (this file now includes the AI settings table too — one file only!)
  4. Note your URL, anon key, service_role key

STEP 3 — Run the setup wizard
  Windows:   Double-click SETUP.bat
  Mac/Linux: bash start.sh

  Or manually:
    cp backend/.env.example backend/.env
    → fill in SUPABASE_URL, SUPABASE_ANON_KEY, SUPABASE_SERVICE_KEY
    cp frontend/.env.example frontend/.env.local

STEP 4 — Start the app
  Windows:   Double-click START.bat
  Mac/Linux: bash start.sh

═══════════════════════════════════════════════════════════════════
ENABLING AI FEATURES  (after the app is running)
═══════════════════════════════════════════════════════════════════

  ⚠️  AI is DISABLED by default. No API key required to start.
  ⚠️  NEVER put your API key in .env — use the Admin Panel instead.

  1. Open: http://localhost:3000/admin
  2. Login as admin@resumeos.com / Admin@123456
  3. Click "AI Settings" tab
  4. Choose a provider:
       • Anthropic (Claude) — console.anthropic.com/settings/keys
       • OpenAI (ChatGPT)   — platform.openai.com/api-keys
       • OpenRouter         — openrouter.ai/keys (100+ models, best value)
  5. Select a model (recommended: claude-3-5-haiku or gpt-4o-mini for cost)
  6. Paste your API key → click Save Settings
  7. Click Test to verify connection
  8. Toggle "AI Features" ON
  9. Done! Users now see AI buttons in the resume builder.

  To disable AI for all users: toggle "AI Features" OFF in Admin Panel.
  Admin can re-enable anytime without restarting the server.
